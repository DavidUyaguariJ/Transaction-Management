# Microservicio de Gestión de Cuentas y Transacciones

## Introducción

Este proyecto corresponde al desarrollo de un microservicio backend para la **gestión de cuentas y transacciones financieras** dentro del contexto de modernización tecnológica de **NovoBanco**.

Este microservicio cubre funcionalidades esenciales como:

- Gestión de cuentas bancarias
- Registro de depósitos y retiros
- Transferencias entre cuentas
- Consulta de historial de movimientos
